# 10-工作流约束升级清单

日期：2026-04-13

## 1. 本次新增能力

1. 优先级调度：`P0 > P1 > P2 > P3`
2. 抢占策略：`Auto-PlanTasks -PreemptLowerPriority` 可回退低优先级进行中的可抢占任务
3. 冲突控制：`Next` 与 `Set-TaskStatus` 均会拦截 `in_progress` scope 冲突
4. 验收标准规范：推荐 `ac.<id>: <condition>` 机器可读格式
5. 重试上限约束：`Set-TaskStatus` 在重试超限时自动转 `blocked`
6. evidence 结构：推荐 `cmd|result|log|artifact` 统一行格式

## 4. 推荐执行顺序

1. `Invoke-Workflow -Action Status`
2. `Auto-PlanTasks -Request "..." -Priority P2`
3. `Invoke-Workflow -Action Next`
4. 子 AGENT 开发与 tester 验证
5. 主 AGENT 回写状态
6. `Invoke-Workflow -Action Validate`
7. `Invoke-Workflow -Action Summary`
