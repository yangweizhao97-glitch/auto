# 文档导航

这份导航用于降低上手成本：先看能跑起来的，再看设计细节，最后看背景材料。

## 先看这三篇（10 分钟入门）

1. [07-成熟版运行手册.md](./07-成熟版运行手册.md)
作用：按命令把工作流跑起来，知道每天怎么用。

2. [09-统一任务拆解方案.md](./09-统一任务拆解方案.md)
作用：理解主/子/测试 AGENT 的职责边界和调度逻辑。

3. [03-tasks-json规范.md](./03-tasks-json规范.md)
作用：理解 `tasks.json` 字段、状态机、依赖规则与最小示例。
